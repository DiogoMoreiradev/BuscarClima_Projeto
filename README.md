Nesse projeto realizado com HTML/CSS3 e Javascript tem a função de procurar climas de cidades por meio do consumo de uma API(Weather API),
Havendo também mudanças no background do index de acordo com a cidade pesquisada por meio de outra API, para utilizar o weather basta fazer
o registro no site https://openweathermap.org/api, entrar no seu perfil e clicar na opção "My API keys".
